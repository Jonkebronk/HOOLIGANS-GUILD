NATO HOOLIGANS Loot Council Platform - Technical Documentation for Claude Code

Overview
This document provides comprehensive specifications for building a World of Warcraft TBC Loot Council Management Platform based on the "NATO HOOLIGANS Loot Council TBC" Excel spreadsheet. The platform manages loot distribution for a raiding guild using a loot council system.

Core Domain Concepts
Loot Council System
A loot council is a method of distributing raid loot where officers evaluate each player's need for items based on criteria like commitment, performance, attendance, reliability, and magnitude of upgrade.

Key Evaluation Criteria

Engagement - Active participation in raids and strategy discussions
Attendance/Reliability - Consistent presence and preparation
Performance - DPS/HPS output, teamwork, and boss mechanics execution
Magnitude of Upgrade - How significant the item is for the player's character (BiS vs minor upgrade)
